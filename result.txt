🎓 สรุปการวิจัย: ระบบแนะแนวแผนกเรียนด้วย Machine Learning

📊 วัตถุประสงค์การวิจัย
พัฒนาระบบแนะแนวอัจฉริยะสำหรับช่วยนักเรียนใหม่เลือกแผนกเรียนที่เหมาะสม โดยเรียนรู้จากข้อมูลนักเรียน ปวช. 1-3 ที่มีอยู่แล้ว

🗂️ ขั้นตอนที่ 1: การเก็บข้อมูล

กลุ่มตัวอย่าง: นักเรียน ปวช. 1,789 คน จาก 3 แผนก

แผนกบัญชี: 504 คน (28.2%)
แผนกสารสนเทศ: 601 คน (33.6%)
แผนกอาหาร: 684 คน (38.2%)


ข้อมูลที่เก็บ: 25 ตัวแปร แบ่งเป็น

คะแนนวิชาหลัก: คณิต, คอมพิวเตอร์, ไทย, วิทยาศาสตร์, ศิลปะ
ทักษะ: การคิดเชิงตรรกะ, ความคิดสร้างสรรค์, การแก้ปัญหา
ความสนใจ: ตัวเลข, เทคโนโลยี, การทำอาหาร
ข้อมูลส่วนตัว: อายุ, เพศ, รายได้ครอบครัว, ฯลฯ




⚙️ ขั้นตอนที่ 2: Data Preprocessing

Normalization: ใช้ Standard Scaler

แปลงข้อมูลให้มี mean = 0, std = 1
ทำให้ทุก features อยู่ในมาตราส่วนเดียวกัน


Train/Test Split: แบ่งข้อมูล 80/20 (1,431/358)
Encoding: เปลี่ยนข้อมูลหมวดหมู่เป็นตัวเลข


🎯 ขั้นตอนที่ 3: Enhanced Feature Selection
วิธีการ: Enhanced Sequential Forward Selection

Domain Knowledge Filtering: กรองตัวแปรตามหลักการทางการศึกษา

ยอมรับ: คะแนน + ทักษะ + ความสนใจ (13 ตัว)
บล็อก: ไลฟ์สไตล์ + รายได้ครอบครัว (8 ตัว) ❌



ผลลัพธ์ Feature Selection:

เริ่มต้น: 21 features → คัดเหลือ: 10 features 📉
Features ที่ถูกเลือก:

คะแนนคณิตศาสตร์ 🔢
คะแนนคอมพิวเตอร์ 💻
คะแนนภาษาไทย 📝
คะแนนวิทยาศาสตร์ 🧪
คะแนนศิลปะ 🎨
ทักษะการคิดเชิงตรรกะ 🧠
ทักษะความคิดสร้างสรรค์ ✨
ความสนใจด้านตัวเลข 📊
ความสนใจด้านเทคโนโลยี ⚡
เพศ 👤


Quality Score: 200/200 ✅ (Academically Defensible!)


🤖 ขั้นตอนที่ 4: Model Training & Evaluation
โมเดลที่ทดสอบ: 3 ตัว

Random Forest 🌲
Gradient Boosting ⚡
Logistic Regression 📈

การประเมินผล:

15-Fold Cross Validation (เข้มข้นสุดๆ!)
Metrics: Accuracy, Precision, Recall, F1-Score
Statistical Testing: T-test, Wilcoxon, ANOVA, Friedman
